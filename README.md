# 心理成熟度测试

基于心理学理论的专业心理成熟度在线测试平台，涵盖八个核心维度的全面评估。

## 功能特性

- **50 道精心设计的题目** —— 35 道 Likert 自评量表 + 15 道情境选择题，约 8-12 分钟完成
- **八维度综合评估** —— 情绪稳定性、自我认知清晰度、责任感与担当、人际成熟度、认知灵活性、冲动控制、共情与视角转换、自我成长驱动
- **多维可视化报告** —— ECharts 雷达图 + 柱状图，直观呈现各维度得分分布
- **深度解读与建议** —— 每个维度提供三段式分析（高/中/低分）及针对性成长建议
- **五级成熟度分类** —— 初始成长 → 探索发展 → 趋于成熟 → 成熟稳定 → 高度成熟
- **隐私安全** —— 所有数据仅存储在浏览器本地，不会上传到任何服务器

## 理论基础

本测试参考以下学术框架设计：

- Kegan 成人发展理论
- Loevinger 自我发展模型
- 中国成年人心理成熟度量表（2009，六维度）
- SAFE 心理成熟度量表（2024）
- Goleman 情绪智力理论

## 快速开始

```bash
# 使用 serve（推荐）
npx serve -p 3456 -s .

# 或使用 Python
python -m http.server 3456
```

浏览器访问 `http://localhost:3456`，无需安装任何依赖。

## 技术栈

- 原生 HTML / CSS / JavaScript（无框架，无构建工具）
- [ECharts 5.5](https://echarts.apache.org/)（CDN 引入）
- 响应式设计，适配 320px - 800px+ 屏幕
